# サブスク整理

> 「気づいたら払い続けていた」をなくす、サブスク整理アプリ。

月額・年額サービスの契約状況を見える化し、見直しや整理をしやすくします。

---

## 想定ユーザー

サブスクが増えすぎて把握しきれていない人、更新日をうっかり見逃しがちな人、月の固定費を整理したい人。

---

## できること

### 登録・管理
- サービス名のサジェスト入力（24サービス収録、カテゴリや金額を自動補完）
- ステータス管理（利用中／見直し中／解約予定／停止済み）
- 解約ページURLの保存とワンタップ遷移
- アーカイブ機能（削除せず一覧から非表示）

### 気づきを促す仕組み
- 更新日超過を赤、7日以内を橙でリスト上に警告表示
- 更新日が過ぎたまま編集すると、次の周期への日付ロールオーバーを提案
- ホームに「要確認（更新日）件数」を常時表示

### 支出の把握
- 月額合計をサマリーバーに常時表示（年額表示に切り替え可）
- カテゴリ別の内訳グラフで支出構成を可視化
- 年払い・3ヶ月払いは月額換算して合算

---

## データについて

端末内のみで完結し、サーバー通信はありません。アカウント登録不要ですぐ使えます。

---
